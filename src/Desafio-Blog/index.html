Introdução

HTML é uma linguagem de marcação padrão para criar páginas web.

Significa Hyper Text Markup Language, ou seja Linguagem de marcação de hiper texto.

Ela descreve a estrutura de uma página da web.

E consiste em uma série de elementos.

Os quais informam ao navegador como exibir o conteúdo.

Exemplo de como os elementos HTML rotulam os fragmentos do contúdo da página informando ao navegador que "Este é um link", "Este é um título", etc.

Elementos da HTML

Exemplos básicos de HTML

Voltar

Um elemento HTML é definido por uma tag inicial, algum conteúdo e uma tag final.

Porém, alguns elementos não possuem conteúdo, como exemplo a tag de quebra de linha.

Esses elementos são chamados de elementos vazios. Elementos vazios não tem uma tag final.

HTML

É o elemento raiz do documento que engloba as seções Head e body.

A tag HTML possui uma tag de abertura e outra de fechamento.

Head

Se destina a marcar a seção HEAD do documento.

Title

Se destina a marcar o título do documento

Body

Se destina a marcar a seção corpo do documento.

Tudo que está dentro desta seção será apresentado pelo navegador ao usuário que solicitou
